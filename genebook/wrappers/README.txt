alignment.php - this wrapper gets orthologs from kegg and submits these to the clustal webservice for alignment, this is then displayed via jalview applet

alignment_buffed.php - this wrapper gets orthologs from kegg and submits these to the clustal webservice for alignment with padding, this is then displayed via jalview applet

Blast_Output.php - gets the results from BLAST EBI webservice and displays them as an SVG file

blast_table.php - parses the results from EBI BLAST webservice into an html table

correlation_wrapper - displays the results from the correlation matrix webservice and parses into an html table

coverage.php - displays the NGS coverage webservice output

Enzyme.php - displays the enzyme from the Expasy Enzyme URL results

eutils_cdd.php - displays a dynamic list of domains from the CDD using the eutilities webservice

gb_parameters.php - the parameters file for wrappers

gene_browser_wrapper.php - displays the genebrowser webservice

gene_loc_graph_wrapper.php - displays the gene location graph from the in house webservice

kegg_orth_table.php - displays the output from the kegg orthology webservice into a table

kegg_ortholog_wrapper.php - displays the results from the kegg ortholog genebrowser

kegg2.php - gets the pathways assocaited with the feature using KEGG webservice, displays the pathways dynamically

NGS_stack.php - this wrapper displays the NGS pile up webservice into an iframe

sequence.php display the sequence from the sequence webservice
